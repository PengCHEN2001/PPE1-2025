Présentation du miniprojet

Ce miniprojet vise à construire les pages HTML :

  • une page d’accueil
  • une page affichant un tableau contenant les résultats de l’analyse des URL collectées.

La page index.html est écrite et stylisée manuellement. La page tableau-fr.html est générée automatiquement par le script miniprojet.sh, qui lit une liste d’URL depuis un fichier texte, et produit un tableau résumant les informations : URL, code HTTP, encodage, nombre de mots.
